# 数据库表结构文档

## 概述

本文档描述了农产品价格分析系统的数据库表结构。数据库采用 PostgreSQL 16 + TimescaleDB。

---

## 数据表清单

| 序号 | 表名 | 说明 |
|------|------|------|
| 1 | users | 用户表 |
| 2 | categories | 分类表 |
| 3 | products | 产品表 |
| 4 | markets | 市场表 |
| 5 | data_sources | 数据源表 |
| 6 | price_records | 价格记录表 |
| 7 | crawl_logs | 爬虫日志表 |
| 8 | price_alerts | 价格提醒表 |
| 9 | alert_logs | 提醒日志表 |

---

## 1. 用户表 (users)

用户账户信息表。

| 字段名 | 类型 | 可空 | 默认值 | 主键 | 外键 | 唯一 | 说明 |
|--------|------|------|--------|------|------|------|------|
| id | uuid | NO | uuid_generate_v4() | ✅ | - | - | 用户ID |
| created_at | timestamp with time zone | NO | now() | - | - | - | 创建时间 |
| updated_at | timestamp with time zone | NO | now() | - | - | - | 更新时间 |
| deleted_at | timestamp with time zone | YES | null | - | - | - | 删除时间（软删除） |
| username | character varying(255) | NO | - | - | - | ✅ | 用户名 |
| password | character varying(255) | YES | null | - | - | - | 密码（加密存储） |
| cellphone | character varying(45) | YES | null | - | - | ✅ | 手机号 |
| state | USER-DEFINED | NO | 'enabled' | - | - | - | 账户状态 |
| is_admin | boolean | NO | - | - | - | - | 是否管理员 |

---

## 2. 分类表 (categories)

农产品分类，支持多级分类。

| 字段名 | 类型 | 可空 | 默认值 | 主键 | 外键 | 唯一 | 说明 |
|--------|------|------|--------|------|------|------|------|
| id | integer | NO | - | ✅ | - | - | 分类ID |
| name | character varying(50) | NO | - | - | - | - | 分类名称 |
| created_at | timestamp with time zone | YES | now() | - | - | - | 创建时间 |
| parent_id | integer | YES | null | - | ✅ (self) | - | 父分类ID |

---

## 3. 产品表 (products)

农产品产品信息。

| 字段名 | 类型 | 可空 | 默认值 | 主键 | 外键 | 唯一 | 说明 |
|--------|------|------|--------|------|------|------|------|
| id | integer | NO | nextval() | ✅ | - | - | 产品ID |
| name | character varying(100) | NO | - | - | - | - | 产品名称 |
| category_id | integer | YES | null | - | ✅ (categories.id) | - | 所属分类 |
| unit | character varying(20) | YES | '公斤' | - | - | - | 计量单位 |
| remark | text | YES | null | - | - | - | 备注 |
| created_at | timestamp with time zone | YES | now() | - | - | - | 创建时间 |

---

农产品批发市场信息。

| 字段名 | 类型 | 可空 | 默认值 | 主键 | 外键 | 唯一 | 说明 |
|--------|------|------|--------|------|------|------|------|
| id | integer | NO | nextval() | ✅ | - | - | 市场ID |
| name | character varying(100) | NO | - | - | - | - | 市场名称 |
| province | character varying(50) | YES | null | - | - | - | 省份 |
| city | character varying(50) | YES | null | - | - | - | 城市 |
| created_at | timestamp with time zone | YES | now() | - | - | - | 创建时间 |

---

## 5. 数据源表 (data_sources)

| 字段名 | 类型 | 可空 | 默认值 | 主键 | 外键 | 唯一 | 说明 |
|--------|------|------|--------|------|------|------|------|
| id | integer | NO | nextval() | ✅ | - | - | 数据源ID |
| name | character varying(50) | NO | - | - | - | - | 数据源名称 |
| url | text | YES | null | - | - | - | 数据源URL |
| crawler | character varying(50) | YES | null | - | - | - | 爬虫类名 |

| created_at | timestamp with time zone | YES | now() | - | - | - | 创建时间 |

---

## 6. 价格记录表 (price_records)

农产品价格时序数据（TimescaleDB 超表）。

| 字段名 | 类型 | 可空 | 默认值 | 主键 | 外键 | 唯一 | 说明 |
|--------|------|------|--------|------|------|------|------|
| time | timestamp with time zone | NO | - | - | - | - | 记录时间 |
| product_id | integer | YES | null | - | ✅ (products.id) | - | 产品ID |
| market_id | integer | YES | null | - | ✅ (markets.id) | - | 市场ID |
| price | numeric | NO | - | - | - | - | 价格 |
| min_price | numeric | YES | null | - | - | - | 最低价 |
| max_price | numeric | YES | null | - | - | - | 最高价 |
| avg_price | numeric | YES | null | - | - | - | 平均价 |
| source | character varying(20) | YES | null | - | - | - | 数据来源 |
| spec_info | character varying(100) | YES | null | - | - | - | 规格信息 |
| unit_info | character varying(20) | YES | null | - | - | - | 单位信息 |

---

## 7. 爬虫日志表 (crawl_logs)

| 字段名 | 类型 | 可空 | 默认值 | 主键 | 外键 | 唯一 | 说明 |
|--------|------|------|--------|------|------|------|------|
| id | integer | NO | nextval() | ✅ | - | - | 日志ID |

| error_msg | text | YES | null | - | - | - | 错误信息 |
| created_at | timestamp with time zone | YES | now() | - | - | - | 创建时间 |

---

## 8. 价格提醒表 (price_alerts)

用户价格提醒配置。

| 字段名 | 类型 | 可空 | 默认值 | 主键 | 外键 | 唯一 | 说明 |
|--------|------|------|--------|------|------|------|------|
| id | integer | NO | nextval() | ✅ | - | - | 提醒ID |
| user_id | uuid | NO | - | - | ✅ (users.id) | - | 用户ID |
| product_id | integer | NO | - | - | ✅ (products.id) | - | 产品ID |
| alert_type | character varying(10) | NO | - | - | - | - | 提醒类型（above/below） |
| threshold_value | numeric | NO | - | - | - | - | 阈值 |
| is_active | boolean | YES | true | - | - | - | 是否启用 |
| created_at | timestamp with time zone | YES | now() | - | - | - | 创建时间 |

---

## 9. 提醒日志表 (alert_logs)

价格提醒触发记录。

| 字段名 | 类型 | 可空 | 默认值 | 主键 | 外键 | 唯一 | 说明 |
|--------|------|------|--------|------|------|------|------|
| id | integer | NO | nextval() | ✅ | - | - | 日志ID |
| alert_id | integer | NO | - | - | ✅ (price_alerts.id) | - | 提醒ID |
| triggered_at | timestamp with time zone | NO | now() | - | - | - | 触发时间 |
| price_value | numeric | NO | - | - | - | - | 触发时价格 |
| threshold_value | numeric | NO | - | - | - | - | 阈值 |
| alert_type | character varying(10) | NO | - | - | - | - | 提醒类型 |
| product_name | character varying(255) | NO | - | - | - | - | 产品名称 |
| is_read | boolean | NO | false | - | - | - | 是否已读 |

---

```

  ↑
  ├── price_alerts → alert_logs
  │
  └── products
        ↑
        └── price_records ← markets
```
